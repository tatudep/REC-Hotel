REC-Hotel

O REC-Hotel é um sistema de gerenciamento para hotéis, desenvolvido em C++. Ele permite o controle eficiente de hóspedes, reservas, quartos, serviços extras e faturamento, tudo por meio de uma interface intuitiva no terminal.

Principais funcionalidades

  • Cadastro e gerenciamento de hóspedes
  • Reserva e controle de quartos
  • Gestão de serviços extras oferecidos pelo hotel
  • Controle de faturamento mensal
  • Interface para administradores e recepcionistas
  • Relatórios de estadias e reservas
